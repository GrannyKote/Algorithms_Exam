# Итоговое домашнее задание по дисциплине "Алгоритмы и структуры данных", вариант 3, Евсюнина М.О.

Проект предназначен для осуществления нечёткого поиска, выявление дублирующихся значений. 
Исходный код проекта написан на языке Python и расположен в файле main.py

Логика работы программы:
- пользователь вводит пути до двух входных файлов и директории, в которой будет размещен выходной файл "duplicates.json"
- если установлен флаг предобработки данных, то программа приводит строки к нижнему регистру и удаляет все символы, не являющиеся числами, буквами или пробелами, и лишние пробелы
- программа находит расстояния Левенштейна для названий товаров из указанных файлов, далее расчитывает коэффициенты схожести. Если коэффициент схожести выше указанного порога, то информация о товаре добавляется в результирующий файл
